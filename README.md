# ChatGPT Me

Moving from ChatGPT to Claude? Common approaches include exporting your memory or asking ChatGPT to summarize you. This app takes a different approach: extract your raw conversation history and let Claude build your profile from the source.

**[Open the app →](https://alyssafuward.github.io/chatgpt-me/)**

## What it does

Reads your ChatGPT export, strips out all the AI responses, and produces a clean file of just your messages — plus a ready-made prompt that asks Claude to analyze how you work and generate a personal system prompt you can reuse across conversations.

## How to use

1. Export your ChatGPT data: **ChatGPT → Settings → Data Controls → Export Data**
2. Open the app and drop in your export (ZIP or JSON files)
3. Set a size limit if you want (default is 10 MB)
4. Click **Extract my conversations**
5. Download your extracted data and the analysis prompt
6. Open Claude, attach both files, and paste the cover message the app provides

Claude will analyze your history and return a working profile and personal system prompt. Review and edit it, combine it with any other profile exercises you've done, then paste it wherever you'd like Claude to use it.

## Privacy

Everything runs locally in your browser. No data is uploaded anywhere.
